HotView Labs

Blog Platform

Project Overview

This project demonstrates my ability to interpret a brief, plan a solution, design a user interface, and build and test a responsive web application. It incorporates the skills learned throughout the FED1 program, including design, HTML, CSS, and JavaScript. The project involves creating a front-end user interface for the Noroff Blog API with responsive features and dynamic content.

Challenge: Create a front-end user interface for the Noroff Blog API that is responsive and includes dynamic content.

Solution: Developed a responsive web application that integrates with the Noroff Blog API, featuring a user-friendly interface and dynamic content display.

Key Features

  • Interactive Banner Carousel: View a rotation of the 3 latest posts with clickable buttons to read more.
  • Responsive Thumbnail Grid: Static list of the 12 latest posts for easy selection.
  • Blog Post Public Page: Displays post details with a shareable URL.
  • Blog Post Edit Page: Allows the owner to edit or delete posts.
  • Create Post Page: Enables the owner to create new blog posts.
  • Account Pages: Register and login forms for the owner.

Design Approach

For the design, I focused on creating a clean and intuitive user interface that resonates with the target audience. Utilizing a modern and minimalist design approach, I aimed to enhance user engagement and ensure ease of navigation throughout the platform.

Color Palette: Utilized a harmonious color scheme to create a visually appealing and cohesive design.

Development Process

The development process involved meticulous planning, efficient coding, and thorough testing. Utilizing industry-standard tools and frameworks, I ensured the project adhered to best practices and met the specified requirements. Continuous iteration and feedback were key to refining the application and delivering a high-quality end product.

Project Highlights

  • Enhanced User Experience: Prioritized a clean design and intuitive navigation to improve user engagement.
  • Dynamic Content Display: Implemented features such as interactive carousels and responsive grids to showcase content effectively.
  • Efficient Development: Utilized modern development tools and methodologies to streamline the development process and ensure timely delivery.
