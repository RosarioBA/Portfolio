My Projects

Below you'll find a selection of my recent projects that demonstrate my skills and expertise in front-end development. Each project highlights my ability to create visually appealing, user-friendly, and functional solutions.

Square Eyes

This project involves creating a video streaming website that serves as a platform for small filmmakers to sell their high-quality videos. The main objective is to develop a responsive and user-friendly website that provides a seamless video purchasing and streaming experience. My role in this project was to design the user interface, develop the website using HTML and CSS, and ensure it meets the needs of the target audience.

Key Features or Achievements:

  • User-Centric Design: Developed a clean and intuitive interface that enhances user experience and facilitates easy navigation.
  • Responsive Layout: Ensured the website is fully responsive, providing a seamless experience across desktops, tablets, and smartphones.
  • Support for Independent Filmmakers: Created a platform that promotes and supports unique films from independent filmmakers.

Oslo Discovery Museum

This project is a website designed for an interactive science museum called the Oslo Discovery Museum, targeting primary and middle school children (ages 7-15) and families with young children. The aim is to excite and entice people to visit the museum with a modern, accessible, and responsive web presence.

Key Features or Achievements:

  • User-friendly navigation: Easy to use for all ages.
  • Responsive design: Adapts to various screen sizes.
  • Hamburger menu for mobile devices: Simplifies navigation on smaller screens.

Hot View Labs

This project demonstrates my ability to interpret a brief, plan a solution, design a user interface, and build and test a responsive web application. It incorporates the skills learned throughout the FED1 program, including design, HTML, CSS, and JavaScript. The project involves creating a front-end user interface for the Noroff Blog API (Noroff Blog API Documentation) with responsive features and dynamic content.

Key Features or Achievements:

  • Interactive Banner Carousel: Implemented a dynamic carousel that displays a rotation of the 3 latest blog posts, allowing users to navigate through posts with clickable buttons for more detailed viewing.
  • Responsive Thumbnail Grid: Designed a responsive layout featuring a grid of 12 latest blog post thumbnails, enabling easy selection and navigation to individual blog posts.
  • Blog Post Edit and Management: Developed admin functionality for the blog post edit page, allowing the owner to log in, create, edit, and delete posts, ensuring efficient content management.
