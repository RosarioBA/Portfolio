Oslo Discovery Museum

Interactive Science Museum Website

Project Overview

This project is a website designed for an interactive science museum called the Oslo Discovery Museum, targeting primary and middle school children (ages 7-15) and families with young children. The aim is to excite and entice people to visit the museum with a modern, accessible, and responsive web presence.

Challenge: Design a new website for an interactive science museum that engages a young audience with a clean layout and a vibrant pastel color palette.

Solution: Developed a user-friendly website for the Community Science Museum with features like responsive design, user-friendly navigation, and SEO enhancements.

Key Features

  • User-Friendly Navigation: Easy to use for all ages.
  • Responsive Design: Adapts to various screen sizes.
  • Hamburger Menu: Simplifies navigation on smaller screens.
  • SEO and Accessibility Enhancements: Ensures better reach and usability.

Design Approach

  • Target Audience: Focused on primary and middle school children (ages 7-15) and families with young children.
  • Color Palette: Used vibrant pastel colors to create an engaging and fun atmosphere.
  • Layout: Emphasized a clean and easily navigable layout.

Development Process

  • Planning: Utilized GitHub's project feature and Kanban board for task management.
  • HTML and CSS Implementation: Built the site using HTML5 and CSS3, following best practices like the DRY principle.
  • Responsive Design: Designed for smaller screens first and used media queries to ensure compatibility across devices.
  • Navigation: Implemented a responsive navigation menu to improve usability on mobile devices.

Testing Process

  • Cross-Browser Testing: Ensured functionality across different browsers.
  • Accessibility Testing: Validated code and checked accessibility for better usability.
  • User Feedback: Gathered feedback from the target audience to identify and fix issues.

Project Highlights

  • Enhanced User Experience: Prioritized a clean design and easy navigation to improve user engagement.
  • Engaging Design: Created a fun and vibrant atmosphere with a pastel color palette.
  • Seamless Navigation: Implemented intuitive navigation to help users find information easily.
